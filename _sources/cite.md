# How to cite

To cite the paper:
```
@article{Hughes_2023,
    doi = {10.3847/1538-3881/acc578},
    url = {https://dx.doi.org/10.3847/1538-3881/acc578},
    year = {2023},
    month = {apr},
    publisher = {The American Astronomical Society},
    volume = {165},
    number = {5},
    pages = {204},
    author = {J. Marcus Hughes and Craig E. DeForest and Daniel B. Seaton},
    title = {Coma Off It: Regularizing Variable Point-spread Functions},
    journal = {The Astronomical Journal}
}
```


To cite the software:
```
@software{marcus_hughes_2023_7853250,
  author       = {Marcus Hughes and
                  Sam Van Kooten},
  title        = {punch-mission/regularizepsf: v0.2.0},
  month        = apr,
  year         = 2023,
  publisher    = {Zenodo},
  version      = {0.2.0},
  doi          = {10.5281/zenodo.7853250},
  url          = {https://doi.org/10.5281/zenodo.7853250}
}
```
